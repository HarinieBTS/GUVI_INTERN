Status Drafter Tool
 
Project Overview
The Status Drafter Tool is a web-based application designed to help employees submit their daily work status in a structured and standardized format. The tool stores the status updates and generates organized reports that can be viewed easily by managers or team leads.
This project helps improve communication, saves reporting time, and ensures consistency in daily progress tracking.

Objective

Provide a structured format for daily status updates
Reduce manual effort in collecting reports
Store and manage employee updates efficiently
Improve productivity and communication within teams

Technologies Used

Node.js
Express.js
SQLite Database
HTML
CSS
JavaScript

Features

Employee status submission form
Automatic report generation
Status data storage using database

Installation & Setup

Open terminal inside project folder and run:
npm install

Run the Application
Start the server using:
node server.js

Open in Browser
http://localhost:3000/index.html

How to Use

Open the application in browser
Enter employee name
Fill daily status details
Completed tasks
Tasks in progress
Blockers
Next day plan
Submit the form
View status reports using report page

System Workflow

Employee enters status details
Data is sent to backend server
Information is stored in database
System generates organized reports
Manager can view reports easily
